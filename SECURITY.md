# Security Policy

## Reporting a vulnerability

**Do not open a public issue.** Send details to the maintainer privately.

DsCode encrypts API keys at rest with AES-256-GCM. Always review debug logs before sharing — they may contain code snippets.

## Supported versions

| Version | Supported |
|---------|-----------|
| `>= 1.0.15` | ✅ Active |
| `< 1.0.15` | ❌ Upgrade required |
